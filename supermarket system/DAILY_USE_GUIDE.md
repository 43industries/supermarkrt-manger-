## üÜò **COMMON ISSUES & FIXES**

### **"Can't add to cart"**
- Check if product has stock
- Red stock number = out of stock

### **"Barcode not working"**
- Click in blue barcode field first
- Check scanner is plugged in
- Try typing manually

### **"M-Pesa won't complete"**
- Did you enter phone number?
- Phone field appears when M-Pesa selected
- Need at least 10 digits

### **"Page won't load"**
- Check server is running
- Bookmark the correct URL
- Try: `http://localhost:8000/system.html`

### **"Lost data"**
- Don't clear browser data
- Data saved in browser
- Use same computer/browser
